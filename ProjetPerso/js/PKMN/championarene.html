Les champions d'arène de Kanto !

Tableau de la liste des badges ainsi que le niveau d'obéissance/Cs utilisable/Stat Augmentée:

Liste des champions d'arènes avec leurs noms ainsi que le type de pokémon joué:

Arène n°1: Pierre d'Argenta

L'Arène d'Argenta, spécialisée dans le type Roche, est simplement un chemin vers le Champion d'Arène Pierre.

Il est possible de contourner le seul dresseur pour l'atteindre.

Arène n°2: Ondine d'Azuria

L'Arène d'Azuria, spécialisée dans le type Eau, est une piscine munie d'un ponton où se trouve notamment le Champion d'Arène, Ondine.

Arène n°3: Major Bob de Carmin sur Mer

L'Arène de Carmin sur Mer, spécialisée dans le type Électrik, est la première à comporter une petite énigme.

Vous devez trouver deux interrupteurs placés dans les quinze poubelles présentes, en sachant que :

- lorsque le premier interrupteur est trouvé, le second sera apparu dans une poubelle adjacente (= à côté)

- si vous ne trouvez pas le second du premier coup, il faudra recommencer et le premier aura changé de place

Astuce: Dès que vous trouvez le premier interrupteur, sauvegardez comme ça si vous vous trompé sur le 2e vous pourrez

reset la partie pour essayer une autre poubelle tout en gardant le premier interrupteur activé.

Arène n°4: Erika de Céladopole

L'Arène de Céladopole, spécialisée dans le type Plante, ressemble à un jardin.

Le Champion d'Arène, Erika, se trouve sur le sol herbu au centre qui n'est accessible qu'avec Coupe.

Arène n°5: Koga de Parmanie

L'Arène de Parmanie, spécialisée dans le type Poison, est un petit labyrinthe fermé par des murs quasi-invisibles.

Prenez simplement le temps de les trouver, et vous arriverez très vite au Champion d'Arène Koga.

Arène n°6: Morgan de Safrania

L'Arène de Safrania, spécialisée dans le type Psy, est composée d'une multitude de pièces reliées entre elles par des téléporteurs.

Pour éviter de vous perdre, prenez uniquement les téléporteurs situés au-dessus/en-dessous de vous :

ainsi, vous explorerez toutes les salles et finirez par rejoindre Morgane.

Arène n°7: Auguste de Cramois'Île

L'Arène de Cramois'Île, spécialisée dans le type Feu, est composée de divers pièces séparées par des barrières.

Arène n°8: Giovanni de Jadielle

L'Arène de Jadielle, spécialisée dans le type Sol, est un labyrinthe de "flèches au sol", qui vous obligent à aller dans une direction lorsque vous marchez dessus.

Atteindre le champion n'est pas vraiment compliqué cependant.

Et voilà vous avez battu les 8 champions d'arène de la région de Kanto !

Il vous reste plus qu'à aller affronter la terrible ligue pokémon qui se trouve sur la droite de Jadielle

Mais juste avant de pouvoir entrer dans la route victoire il vous restera un adversaire à affronter et qui vous

bloquera la route, il s'agit tout simplement de votre Rival avec un niveau assez elevé pour aller affronter la ligue.
